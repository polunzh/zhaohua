# Zhaohua (朝花夕拾)

A web-based life simulation set in 1990s rural China. Experience the daily life of a village primary school teacher and a town postman.

## Features

- **Real-time world** — Game time syncs with real time. The world keeps running when you're away.
- **Two character lines** — Play as a teacher or a postman, switch anytime.
- **12 locations** — School (classroom, playground, flower pool, water tower, office), village (roads, farmland, houses), town (post office, market, clinic).
- **NPC system** — Characters with schedules, moods, and relationships that respond to your choices.
- **Event system** — Daily events, seasonal events, and long-form story arcs.
- **AI dialogue** — NPC conversations generated by AI with 1990s rural Chinese flavor. Supports DeepSeek, Kimi, MiniMax, OpenAI, Claude.
- **Nostalgic visuals** — 90s film color grading inspired by Kodak Gold 200, hand-drawn illustration style.

## Tech Stack

- **Frontend:** Vue 3, Canvas API
- **Backend:** Nitro
- **Database:** SQLite (better-sqlite3)
- **Toolchain:** VitePlus (Vite + Rolldown + Vitest + Oxc)
- **AI:** Multi-model adapter (OpenAI-compatible + Anthropic)

## Getting Started

```bash
# Install dependencies
pnpm install

# Configure AI
cp .env.example .env
# Edit .env with your API key

# Start development
pnpm dev          # Frontend (port 5173)
pnpm dev:server   # Backend (port 3001)

# Run tests
pnpm test
```
