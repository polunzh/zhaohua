# 朝花夕拾

像素风 RPG 生活模拟，体验 1990 年代中国北方农村的日常。画面参考星露谷物语，扮演一名村小学老师或镇上邮递员，在小小的乡村社区里过日子。

## 功能

- **像素 RPG 画面** — 星露谷风格瓦片地图，柯达 Gold 200 色调，四季与天气视觉变化。
- **实时世界** — 游戏时间与现实同步，你不在的时候世界继续运转。
- **双线体验** — 老师线和邮递员线，随时切换。
- **16 个场景** — 学校、村庄、镇上、学生家，每个场景都有手工绘制的瓦片地图。
- **NPC 互动** — 有日程、有情绪、有关系。上下文选项菜单：检查作业、鼓励学生、和村民聊天等。
- **每日任务** — 每天一个新目标，引导你探索不同场景。
- **礼物收集** — NPC 会根据你的选择和关系送你礼物。
- **旅途事件** — 在场景间移动时随机触发，随季节和天气变化。
- **连续签到** — 每日登录奖励，3、7、14、30 天里程碑提示。
- **学生成绩与关系面板** — 查看每个学生的学习进度和与所有 NPC 的关系。
- **AI 对话** — NPC 对话由 AI 生成，带有 90 年代农村味道。支持 DeepSeek、Kimi、MiniMax、OpenAI、Claude。
- **Toast 通知** — 任务完成、收到礼物、连续签到等即时反馈。
- **224 个单元测试 + 14 个 e2e 测试** — 全面的测试覆盖。

## 技术栈

- **前端：** Vue 3、Canvas API
- **后端：** Nitro
- **数据库：** SQLite (better-sqlite3)
- **工具链：** VitePlus (Vite + Rolldown + Vitest + Oxc)
- **AI：** 多模型适配器（OpenAI 兼容 + Anthropic）

## 快速开始

```bash
# 安装依赖
pnpm install

# 配置 AI API Key
echo 'AI_API_KEY=your-key-here' > .env.local

# 启动开发
pnpm dev          # 前端 (端口 5173)
pnpm dev:server   # 后端 (端口 3001)

# 运行测试
pnpm test
```
