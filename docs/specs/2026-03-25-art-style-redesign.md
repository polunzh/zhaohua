# 画风重做：卡通矢量风格

## 决策

- 场景：整幅 SVG 矢量场景，每个地点一个 Vue 组件
- 角色：Q 版大头，黑色粗描边（2-3px），大眼睛
- 色调：温暖褪色旧时光色板（低饱和、暖黄）
- 技术：SVG + CSS 替换 Canvas/PixiJS

## 色板

保持现有旧时光色板，但降低饱和度适配卡通风：

- 草地：#8a9a78 → #9aaa88（稍亮但不鲜艳）
- 泥土：#c4b088
- 砖墙：#b8786a
- 天空/背景：#d8d0b8
- 描边：#4a4040（深棕，不是纯黑）
- 肤色：#d4b898
- 树叶深：#6a8a58
- 树叶浅：#a8c490

## 场景结构

每个场景是一个 Vue 组件：

- 固定 SVG 背景（地面、建筑、植物）
- NPC 位置预设，可点击
- 出口区域可点击
- 季节/天气通过 CSS 滤镜变化

## 角色设计

SVG 矢量角色，每个约 80×120px：

- 头占 40%（大圆 + 发型）
- 眼睛大而有神（椭圆 + 高光点）
- 身体小（矩形 + 圆角）
- 所有边缘 2-3px 深棕描边
- 不同角色通过发型、发色、衣服颜色区分

## 迁移策略

1. 创建 SVG 场景组件替换 PixiGameCanvas
2. 创建 SVG 角色组件替换 drawCharacter
3. App.vue 切换到新组件
4. 保留所有游戏逻辑不变
