# 像素美术重做 — 参考星露谷风格

## 目标

角色和场景全部参考星露谷物语的像素美术风格重做。

## 角色

- 尺寸：16×32 像素（占 1×2 tile）
- 比例：Q 版大头，头占 ~12px 高（接近 1/3 身高）
- 圆脸、大眼睛（2×2 或 3×2 像素的眼睛）
- 明显的发型差异（短发、长发、扎辫子等）
- 衣服颜色鲜明但在旧时光色板内
- 有小嘴巴、腮红等可爱细节
- 4 方向面朝（目前只需 down 方向，面向玩家）

## 场景 Tile（16×16）

参考星露谷的 tile 特点：

- **地面**：不是纯色，有纹理变化（草地有深浅色块交错、土地有颗粒感、地板有砖缝线）
- **墙壁**：有砖缝、有上下阴影渐变、有高光
- **物体**：有立体感（桌面高光、侧面阴影）、比例协调
- **自然**：树冠有 3-4 种绿色层次、花朵有花瓣细节、竹子有节和叶
- **整体**：每个 tile 用 6-10 种颜色，不是 2-3 种

## 色板

保持旧时光褪色暖调，但在此基础上丰富每个 tile 的颜色层次：

- 每种基础色有 light / base / dark / shadow 四个层级
- 草地：#8aac86 / #7A9178 / #5a7a58 / #4a6a48
- 土地：#dcd0a8 / #D4C08E / #C9A882 / #b89870
- 砖墙：#d88a84 / #C4706A / #a05a54 / #8a4a44
- 木头：#a08030 / #8B6914 / #6B5B4E / #5a4a3e
